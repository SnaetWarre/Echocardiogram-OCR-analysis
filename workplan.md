# Work Plan – Internship Project

## Background & Context

The goal of this project is to develop an AI-powered tool that automatically extracts structured measurement data from echocardiogram images. Currently, clinicians and researchers must manually open and review large volumes of echocardiogram files to retrieve relevant measurements, which is a time-consuming and error-prone process. This project aims to solve that problem by applying Optical Character Recognition (OCR) and AI techniques to automatically detect and read on-image annotations from echocardiogram files. The extracted data will then be structured and made available in a usable format, eliminating the need to open each image individually.

## Main Goals & Tasks

- Research and select suitable AI and OCR technologies for echocardiogram image analysis
- Build a pipeline that processes echocardiogram images and automatically extracts relevant text and measurements
- Validate the extracted results for accuracy and reliability
- Develop a user interface to review and export the structured data

## Tentative Timeline (≈ 3 months)

### Month 1 – Research & Orientation
Study the echocardiogram image format, identify the relevant measurement fields, and evaluate available OCR and AI approaches. Define the scope and technical requirements of the project.

### Month 2 – Development
Implement the core extraction pipeline, integrate the chosen OCR/AI solution, structure the output data, and connect it to a basic interface for review and export.

### Month 3 – Testing, Validation & Refinement
Run the tool on a larger set of images, measure accuracy, address edge cases and failure scenarios, and finalize documentation.
